 # Mobile Sales Dashboard Analysis Using Power BI

This project focuses on analyzing mobile sales data using Power BI. It demonstrates an end-to-end data analysis workflow from data cleaning to interactive dashboard creation, helping in extracting meaningful business insights.

 ## Dataset Used
<a href="https://github.com/Tamana14/Mobile_Sales_Analysis_Dashboard/blob/main/Mobile%20Sales%20Data.xlsx">Dataset</a>

## Tools & Technologies

### Tools & Technologies
- Excel  
- Power Query  
- Power BI  
- DAX (Data Analysis Expressions)  

###  Workflow
- Cleaned and transformed raw data using Power Query  
- Created DAX measures (Total Sales, Quantity, Avg Price)  
- Built an interactive dashboard in Power BI  
- Used slicers for filtering (city, brand, month, payment method)

 ### Dashboard Interaction
<a href="https://github.com/Tamana14/Mobile_Sales_Analysis_Dashboard/blob/main/Screenshot%202026-04-18%20003847.png">View Dashboard</a>


###  Key Insights
- Identified sales trends and top-performing models  
- Analyzed customer ratings and payment methods  
- Generated insights for better decision-making  

###  Features
- Interactive dashboard  
- Dynamic filtering using slicers  
- KPI tracking (sales, quantity, transactions, avg price)

### Conclusion

This project highlights how data can be transformed into meaningful insights using Power BI, enabling better understanding of sales performance and customer behavior.

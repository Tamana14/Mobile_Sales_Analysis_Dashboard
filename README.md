 # Mobile Sales Dashboard Analysis Using Power BI

This project focuses on analyzing mobile sales data using Power BI. It demonstrates an end-to-end data analysis workflow from data cleaning to interactive dashboard creation, helping in extracting meaningful business insights.

## Dataset Used
<a href="https://github.com/Tamana14/Mobile_Sales_Analysis_Dashboard/blob/main/Mobile%20Sales%20Data.xlsx">Dataset</a>

## Tools & Technologies
Microsoft Excel
Power Query
Power BI
DAX (Data Analysis Expressions)

## Project Workflow
Data Cleaning & Transformation
Cleaned and transformed raw data using Power Query
Handled missing values and ensured data consistency
Data Modeling & DAX
Created calculated measures using DAX (Total Sales, Total Quantity, Average Price)
Structured data for efficient analysis
Dashboard Development
Built an interactive dashboard in Power BI
Added slicers for dynamic filtering (City, Brand, Month, Payment Method)
